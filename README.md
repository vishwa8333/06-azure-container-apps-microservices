# Azure Container Apps Microservices Platform

## Goal

Deploy a microservices platform using Azure Container Apps, Dapr, internal service discovery, Key Vault, and autoscaling.

## Azure Services

- Azure Container Apps
- Azure Container Registry
- Azure Key Vault
- Azure Service Bus
- Azure Monitor
- Log Analytics Workspace

## DevOps Skills Demonstrated

- Serverless containers
- Microservice deployment
- Dapr service invocation and pub/sub
- KEDA autoscaling
- Managed identity
- CI/CD to Azure Container Apps
